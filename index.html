02

차량이 어느 배차자 건인지 헷갈립니다

차량번호만 보고 어느 원배차자 거래인지 매번 확인해야 합니다.

03

미수 잔액을 즉시 파악하지 못합니다

거래처별로 얼마를 받아야 하는지 장부를 뒤지기 전엔 모릅니다.

04

설치형 프로그램은 PC에 묶여 있습니다

그 PC 앞에 앉아야만 일할 수 있고, PC가 고장 나면 자료가 위태롭습니다.

05

누가 무엇을 입력했는지 권한이 불분명합니다

소장·사무실·관리자 역할 구분 없이 같은 화면을 함께 써 혼선이 생깁니다.

수납록이 해결합니다

한 화면에서 끝나는 6가지

차량번호로 원배차자 자동 인식

차량번호를 입력하면 등록된 원배차자가 자동으로 연결됩니다. 어느 거래 건인지 매번 확인하지 않아도 됩니다.

현장↔사무실 실시간 공유

소장이 현장에서 입력한 계근이 사무실 화면에 실시간(Realtime)으로 나타납니다. 여러 사용자가 동시에 같은 데이터를 봅니다.

건별 입금 — 현금·계산서·카드

계근 건마다 입금을 등록합니다. 현금·세금계산서·카드 유형을 구분해 기록하므로 어떤 건이 정산됐는지 명확합니다.

거래원장·미수원장 + 엑셀

거래원장과 미수원장이 자동으로 정리되고 엑셀로 내보낼 수 있습니다. 60일이 넘은 미수는 경보로 표시돼 놓치지 않습니다.

SMS 템플릿·발송이력·재발송

청구 SMS 템플릿을 저장해두고 발송하며, 발송 이력이 남고 실패한 건은 재발송할 수 있습니다.

역할별 권한 — 소장·사무실·마스터

소장(현장)·사무실·마스터(관리자) 역할로 권한을 나눠 부여합니다. 담당에 맞는 화면과 작업만 노출됩니다.

왜 믿어도 되는가

처리장 데이터를 안전하게 다룹니다

데이터 격리

사업장별 데이터 격리 (RLS)

모든 데이터는 사업장 단위로 격리(Row Level Security)됩니다. 다른 사업장이 우리 데이터에 접근할 수 없습니다.

백업

관리형 클라우드 백업

데이터는 관리형 클라우드에 자동으로 백업됩니다. PC 고장이나 분실로 자료를 잃을 걱정 없이 어느 기기에서나 접속합니다.

감사

계근은 수정 불가, 취소만 가능

입력된 계근은 수정되지 않습니다. 사유 입력 후 취소만 허용해, 미수 분쟁 시 추적 가능한 기록이 남습니다.

시작 전 궁금한 점

프로그램 설치가 필요한가요?
설치가 필요 없습니다. 웹 브라우저에서 가입 즉시 바로 사용할 수 있고, 별도 설치 파일이나 PC 사양 제약이 없습니다.
무료로 먼저 써볼 수 있나요?
14일 무료체험을 제공합니다. 체험 기간에는 모든 기능을 그대로 쓸 수 있으며, 결제 카드 없이 시작할 수 있습니다.
SMS 발송은 어떻게 설정하나요?
SMS 발송은 CoolSMS와 직접 계약한 뒤 발급받은 API 키를 사업장 설정에 입력하면 됩니다. SMS 발송 비용은 CoolSMS에 직접 부과되며 수납록 구독료와는 별도입니다.
데이터는 안전하게 보관되나요?
사업장별로 데이터가 격리(RLS)되어 다른 사업장이 접근할 수 없습니다. 데이터는 관리형 클라우드에 자동으로 백업되며, 계근 기록은 수정이 불가능하고 사유 입력 후 취소만 가능해 분쟁 시 추적할 수 있습니다.
해지하거나 환불받을 수 있나요?
월 단위 구독으로 약정이 없으며, 언제든 다음 결제 전에 해지할 수 있습니다. 우선 14일 무료체험으로 충분히 검토한 뒤 결제 여부를 결정하시면 됩니다.
플랜은 어떻게 고르나요?
모든 플랜의 기능은 동일하고, 차이는 동시에 사용할 수 있는 사용자 수입니다. 스타터 3명, 스탠다드 5명, 프로 10명이며 이용 중에 변경할 수 있습니다.
