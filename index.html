수납록이 해결합니다

한 화면에서 끝나는 6가지

계근 입력 → SMS 자동

차량번호 입력 시 운반자·기본 공차중량·단가가 자동으로 채워집니다. 저장과 동시에 배출자에게 청구 SMS가 발송돼요.

완전 오프라인 동작

인터넷이 끊겨도 계근·장부 작업이 계속 됩니다. SMS 발송 시에만 인터넷을 사용해요. 현장 PC가 인터넷 신호에 흔들리지 않습니다.

월정산서 5분 일괄 발송

결제 주기 도래한 거래처를 자동으로 추려줍니다. 확인 후 버튼 한 번이면 전체 발송. 30곳도 5분이면 끝.

입금 매칭 · 자동 장부

입금을 정산서에 FIFO로 자동 분배해요. 미수원장·거래원장이 실시간으로 정리되고 엑셀로 내보낼 수 있습니다.

SMS·LMS 단가 실시간 표시

발송 전 본문 길이로 SMS 18원인지 LMS 45원인지 사전 안내. 월 SMS 비용을 예측할 수 있습니다.

자동 백업 · 14일 무료

24시간마다 일관 백업으로 자료 손실 위험 차단. PC가 바뀌어도 백업 파일 복사로 그대로 살아납니다. 14일 무료로 먼저 써보세요.

자주 묻는 질문

설치 전 궁금한 점

인터넷 없이도 되나요?
계근·청구·정산·장부 모두 오프라인으로 동작합니다. SMS 발송 시에만 인터넷을 사용해요. 인터넷이 끊겨도 작업이 멈추지 않습니다.
PC 1대만 써야 하나요?
기본 라이선스는 PC 1대 영구입니다. 두 번째 PC가 필요하면 추가 19만원. 같은 사업장이라면 로컬 네트워크로 DB를 공유할 수 있습니다.
데이터는 어디 저장되나요?
사용자 PC의 %APPDATA%\com.dygks.sunaprok\sunaprok.db 한 곳에만 저장됩니다. 외부로 전송되지 않습니다.
SMS 비용은 따로 드나요?
네. 솔라피(Solapi)와 직접 계약하시면 됩니다. SMS 18원 / LMS 45원이며 앱이 발송 전에 단가를 미리 표시해줍니다. 월 청구 200건이면 약 3,600원, 정산 30건(LMS)이면 1,350원입니다.
백업은 어떻게 하나요?
백업 폴더(외장디스크·네트워크 폴더)를 지정한 뒤 수동 버튼으로 즉시 백업하거나 24시간마다 자동 백업이 돕니다. SQLite VACUUM INTO 방식이라 운영 중에도 일관된 스냅샷이 떠집니다.
환불 · 체험은 가능한가요?
14일 무료 체험이 제공됩니다. 체험 기간엔 모든 기능을 그대로 쓰실 수 있어요. 마음에 들면 라이선스 키를 등록하시면 됩니다.
업데이트는 무료인가요?
1년간 무상 업데이트 + 전화·원격 지원이 포함됩니다. 이후엔 연 5만원으로 갱신할 수 있으며, 갱신 안 하셔도 기존 기능은 그대로 사용 가능합니다.
Mac에서도 되나요?
v1은 Windows 10/11 전용입니다. Mac 지원은 향후 검토 항목입니다.
PC 바꾸면 데이터 옮길 수 있나요?
백업 파일(.db)을 새 PC의 같은 위치로 복사하면 그대로 사용할 수 있습니다. 라이선스 키만 재입력하시면 됩니다.
솔라피 가입 절차가 복잡한가요?
솔라피 콘솔에서 발신번호 사전 인증을 1회(약 1주) 진행하시고 API 키를 받으시면 됩니다. 받은 키를 앱 설정 → SMS 입력 칸에 붙여넣으면 끝.
